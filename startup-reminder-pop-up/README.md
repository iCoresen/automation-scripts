一个精美的开机激励工具，在系统启动时显示炫酷的动画弹窗，帮助你以积极的心态开始新的一天。

## 🎨 视觉特效

- **深空渐变背景** - 从深紫到深蓝的渐变色背景
- **流星雨动画** - 随机生成的流星划过屏幕

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Windows / macOS / Linux（内置 tkinter 支持）

### 运行脚本
```bash
python motivation_window.py
```

### 设置开机自启动

**Windows 方法1：启动文件夹**
```
1. Win+R 输入: shell:startup
2. 创建快捷方式指向脚本
3. 右键快捷方式 → 属性 → 目标
   修改为: pythonw.exe "完整路径\motivation_window.py"
```

**Windows 方法2：任务计划程序**
```
1. 打开"任务计划程序"
2. 创建基本任务 → 触发器选择"登录时"
3. 操作选择"启动程序"
4. 程序选择 pythonw.exe
5. 参数填写脚本路径
```

**macOS 方法：**
```bash
# 创建 plist 文件
~/Library/LaunchAgents/com.motivation.window.plist
```

**Linux 方法：**
```bash
# 添加到 ~/.config/autostart/
# 或编辑 ~/.bashrc 添加启动命令
```

## 🛠️ 技术实现

### 核心技术
- **tkinter** - Python 内置 GUI 库
- **Canvas 绘图** - 自定义图形和动画
- **多线程** - 实现自动关闭功能
- **数学计算** - 流星轨迹和粒子运动

### 动画原理

**流星效果**
```
1. 随机生成流星起点和角度
2. 按速度计算位移（cos/sin 三角函数）
3. 绘制渐变尾迹（分段绘制，透明度递减）
4. 出屏幕后重置位置
```

**粒子效果**
```
1. 初始化随机位置和速度
2. 每帧向上移动
3. 到达顶部后重置到底部
4. 使用半透明蓝白色
```

## ⚠️ 注意事项

- **字体兼容性**：Windows 使用 "Microsoft YaHei UI"，其他系统可能需要修改字体
- **无边框窗口**：某些 Linux 桌面环境可能不支持 `overrideredirect`
- **后台运行**：使用 `pythonw.exe`（Windows）可避免弹出命令行窗口
- **性能影响**：动画较多，低配设备可能有轻微卡顿

## 🔧 常见问题

**Q: 窗口不显示？**  
A: 检查是否被其他全屏窗口遮挡，或尝试去掉 `attributes('-topmost', True)`

**Q: 中文显示乱码？**  
A: 脚本文件保存为 UTF-8 编码

**Q: 如何禁用开机自启？**  
A: 删除启动文件夹中的快捷方式或任务计划

**Q: 流星动画不流畅？**  
A: 调整 `animate_meteors` 中的 `after` 延迟值（默认30ms）

**Q: 想要更多流星？**  
A: 修改 `init_meteors` 中的循环次数（默认3个）
